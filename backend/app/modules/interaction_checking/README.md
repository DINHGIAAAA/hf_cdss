# Interaction Checking

Checks drug-drug and drug-condition interaction warnings.

Week-7 scope focuses on high-value HFrEF medication safety interactions.

Implemented checks:

- ACE inhibitor + ARB combination.
- RAAS-inhibiting therapy + MRA potassium/renal monitoring.
- RAAS-inhibiting therapy + NSAID renal safety review.
- Anticoagulant + antiplatelet bleeding risk review.
